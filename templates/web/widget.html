💪 BODYBALANCE.AI

Embed an AI-powered chatbot on your website in minutes

🚀 Quick Integration

Add these two lines to your website's HTML, just before the closing </body> tag:


<!-- BODYBALANCE.AI Widget --> <script src="https://your-domain.com/widget.js"></script> <script> BodyBalanceWidget.init({ apiUrl: 'https://your-api-domain.com', position: 'bottom-right', primaryColor: '#2E7D32', title: 'Support Chat', greeting: '👋 Hi! How can I help you today?' }); </script>

📋 Setup Steps

Deploy the API

Deploy backend_api.py to a hosting service (Render, Railway, Fly.io, etc.)

Host the Widget Script

Upload widget.js to your CDN or serve it from your API server

Add to Your Website

Copy the integration code above and paste it into your website's HTML

Customize

Adjust colors, position, and greeting to match your brand

⚙️ Configuration Options

Option Type Default Description
apiUrl string required Your API server URL
position string 'bottom-right' 'bottom-right' or 'bottom-left'
primaryColor string '#2E7D32' Brand color (hex)
title string 'BODYBALANCE.AI' Widget header title
greeting string '👋 Hi!...' Initial greeting message
